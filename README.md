# Hysteresis Test Results Appendix

This repository contains the appendix tables associated with the article reporting hysteresis test results for composite disc-spring assemblies with and without infill material.

## Repository contents

- `appendix/appendix_tables.pdf`: single PDF file containing all appendix tables.
- `data/`: CSV files separated by table, plus `all_tables_combined.csv`.
- `docs/variable_description.md`: description of variables and units.
- `LICENSE`: license information for reuse of the material.
- `CITATION.cff`: citation metadata for the repository.

## CSV files

| File | Description |
|---|---|
| `data/table_01_VB_0-90-0-90_without_infill.csv` | Table 1. Results obtained from hysteresis for loading of 2831N without infill |
| `data/table_02_CM_0-90-0-90_without_infill.csv` | Table 2. Results obtained from hysteresis for loading of 2681N without infill |
| `data/table_03_CM_90-0--45-45_without_infill.csv` | Table 3. Results obtained from hysteresis for loading of 2660N without infill |
| `data/table_04_VB_0-90-0-90_with_infill.csv` | Table 4. Results obtained from hysteresis for loading of 2831N with infill |
| `data/table_05_CM_0-90-0-90_with_infill.csv` | Table 5. Results obtained from hysteresis for loading of 2681N with infill |
| `data/table_06_CM_90-0--45-45_with_infill.csv` | Table 6. Results obtained from hysteresis for loading of 2660N with infill |
| `data/all_tables_combined.csv` | Combined dataset containing all tables in one file. |
